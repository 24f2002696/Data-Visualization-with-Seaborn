# Customer Engagement Correlation Matrix

**Email:** 24f2002696@ds.study.iitm.ac.in

## Description
This repository contains a Seaborn heatmap visualization analyzing correlations between different customer engagement metrics for Kshlerin Lakin and Green's retail client.

## Metrics Analyzed
- Website Visits
- Email Opens
- Purchase Frequency
- Customer Support Interactions
- Social Media Engagement
- App Usage

## Tools Used
- Python 3.x
- Seaborn
- Matplotlib
- Pandas
- NumPy

## Files
- `chart.py` - Python script to generate the visualization
- `chart.png` - Generated heatmap (512x512 pixels)
- `README.md` - This file

## Running the Script
```bash
python chart.py
```

This will generate `chart.png` showing the correlation matrix heatmap.
